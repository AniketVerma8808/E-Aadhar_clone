What is Aadhar VID?

What is VID Number?

VID number is Virtual ID, it is a 16 digit random number that is revocable. The VID number is mapped with the aadhaar number and can be regenerated many times by the holders. The VID number is valid for one day, it is used for online KYC and authentication transactions. The Aadhar VID number can be used instead of giving your aadhaar number for any online verification or services.

Features of Aadhar VID:

  • An individual can generate the aadhaar VID number without any limits. The only limit is that an individual can generate the Virtual ID again after 24 hours of generating the ID.
  • VID number will be valid till the individual user does not generate the new virtual ID.
  • The Aadhar virtual ID was started to prevent the misuse of the personal details of an individual. Agencies don’t have permission to store or copy the data of individuals aadhaar.
  • Aadhar VID number can be generated by the aadhaar number holder only.

What is the use of Aadhar Virtual ID?

A VID allows a user to go through the Aadhaar biometric ID program verification process without exposing their 12-digit Aadhaar ID number to other parties. When validating an individual's identification, a VID can be used in place of their Aadhaar number. It is possible to undertake authentication using the VID in the same way as Aadhaar is done after the 16-digit number is produced. According to the Unique Identification Authority of India (UIDAI), the body in charge of providing the Aadhaar number and card, it is not feasible to obtain the Aadhaar number from a VID.

  • Only one VID will ever be associated with a single Aadhaar number. Benefits of Using Aadhaar Virtual Id:

  1. Access to the necessary information. This safeguards the crucial information from being misused.
  2. As a result, government entities would have less work to do in terms of obtaining Aadhaar numbers and may use them for KYC instead.
  3. Because Virtual ID is revocable, there is very little likelihood of a duplicate being created.
